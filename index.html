METRICULOUS

Downlaod me
Stats on your code repository, packaged as a java microservice.
Lightweight embedded web server and user interface.
java -jar metriculous [your-local-git-dir] [port]

Download JAR
java -jar metriculous '/Users/yourName/myRepo/.git' 8080
or on windows
java -jar metriculous 'C:/yourName/myRepo/.git' 8080

Alternatively, a full settings and license file (human readable) can be placed in your home directory.
Selected Features
  • Most productive coders - number of lines in production, percentage of code that stays in production without being refactored by someone else.
  • Common conflicts - Identify issues within team, most frequent conflicts and refactorings between people.
  • Hotspots - Files that are most commonly edited, have frequent conflicts.
  • Calendar - when is the work getting done.
  • Simple - most lines, most commits, etc.
  • Pulse on steroids
All presented with simple to read graphs and charts, and a http api for direct access to data.
